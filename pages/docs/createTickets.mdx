# Create Tickets

After you created the event you can use the event id for querying and to create tickets on the event.

```graphql
query ($eventId: ID!, $quantity: Int!, $ticketType: TicketTypeInput!) {
  node(id: $eventId) {
    ... on TicketedEvent {
      createNewTicketType(quantity: $quantity, ticketType: $ticketType) {
        id
        signerPayload
        expectedSigningAddress {
          address
        }
      }
    }
  }
}
```

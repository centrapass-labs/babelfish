# Create Event

As a event organizer the first step is to create the ticketed event.
An ticketed event is stored on the blockchain as a NFT collection.

To interact with the blockchain you first need to create the correct payload to sign with the events organizers private keys.
In this case you create Ticketed Events on the address that is associated with the events organizers private key.

The follow query is an example of how this can be done.

### Query

```graphql
query ($address: String!, $eventDetails: TicketedEventDetailsInput!) {
  network(network: CENNZnet_Nikau) {
    address(address: $address) {
      createTicketedEvent(eventDetails: $eventDetails) {
        id
        signerPayload
      }
    }
  }
}
```

We then need to sign the signerPayload with the correct private key, and submit the signature along with the transcation ID.

### Query

```graphql
mutation ($transactionId: ID!, $signature: String) {
  submitTransaction(transactionId: $transactionId, signature: $signature) {
    status
    result {
      ... on TicketedEvent {
        id
        name
      }
    }
  }
}
```

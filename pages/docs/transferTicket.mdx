After you have created the tickets you can transfer an indiviual Ticket by query on it.

## Query

```graphql
query ($ticketId: ID!, $toAddress: String!) {
  node(id: $ticketId) {
    ... on Ticket {
      createTransferTransaction(toAddress: $toAddress) {
        id
        signerPayload
        expectedSigningAddress {
          address
        }
      }
    }
  }
}
```

We then need to sign the signerPayload with the correct private key, and submit the signature along with the transcation ID.

```graphql
mutation ($transactionId: ID!, $signature: String) {
  submitTransaction(transactionId: $transactionId, signature: $signature) {
    status
    result {
      id
      ... on Ticket {
        name
        owner {
          address
        }
      }
    }
  }
}
```
